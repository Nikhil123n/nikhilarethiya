# Sketchbook Portfolio Artifacts

This folder contains deploy-safe website artifacts used by the sketchbook portfolio. Source design sheets, contact sheets, typography wordmarks, and other unused reference exports are intentionally excluded from the public repository.

## What Is Included

- `01-homepage-layout/` - deploy-used homepage layout cutouts only.
- `03-icons-doodles/` - hand-drawn icons, doodles, arrows, scribbles, cursor, and tech icons.
- `04-hand-drawn-ui-elements/` - sticky notes, paper notes, tape, pushpins, bubbles, buttons, arrows, and paperclip.
- `06-color-palette/` - color tokens, CSS variables, JSON, and swatch assets.
- `07-ui-cards-components/` - project card, skill badge, sticky note, section title, and journey step card assets.
- `08-textures-background/` - paper, notebook, torn paper, tape, hatching, grid, and watercolor texture assets.
- `09-buttons-nav-tabs/` - colored nav tabs and CTA buttons.
- `09-toolkit-strip-icons/` - combined SVG sprite for the toolkit strip.

Use SVG versions where scalable UI is needed. Use transparent PNG cutouts only where the live page references them directly.
